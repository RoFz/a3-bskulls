# Security Policy

## Supported Versions

This repository does not currently maintain multiple supported release lines.
Security fixes are applied to the latest versions of the addons in this repository.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Until private vulnerability reporting is enabled on the public repository, report
potential vulnerabilities by contacting the maintainer directly through GitHub.
Include:

- a clear description of the issue
- affected addon(s) and version(s)
- reproduction steps or proof of concept
- expected impact

I will acknowledge receipt as soon as practical, confirm whether the issue is in
scope, and work toward a fix before any coordinated public disclosure.
